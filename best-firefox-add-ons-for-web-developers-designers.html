Best 5+ Firefox add-ons for web developers and designers

Here's a list of some of the most useful Firefox extensions: the top 5 for web developers and designers, 3 for everyone, and a few more untested - but possibly useful - extensions.

  • 5 ESSENTIAL Firefox extensions for Developers
  • 3 Firefox extensions for EVERYONE
  • Untested Firefox extensions for Developers
  • For More Information

 

 

 

5 ESSENTIAL Firefox extensions for Developers

  • Firebug - trace the CSS impacting each element; change it and view changes in browser
  • Fireshot - capture the visible screen or entire page and edit/save the image: particularly great for before and after shots of changes you're making to sites
  • IE Tab - open a page in an Internet Explorer tab and test your designs for IE browser, but requires you to have IE installed on your computer
  • Web Developer toolbar - see web site sans CSS; outline elements; alter CSS on the fly; view in small screen mode; show alt tags; remove images; use the ruler to measure the size of areas; validate code; get site color scheme in hex values
  • Yellowpipe Lynx Viewer - right-click and see a text-only version of your page: helpful for determining accessibility and for seeing your pages the way search engines do.

3 Firefox extensions for EVERYONE

  • Delicious bookmarks - bookmarks that are accessible anywhere via the web, and can be synced in multiple web browsers on multiple computers. I particularly enjoy the favorite tags feature in my tool bar
  • FlashGot - download all or selected links/files from a web page with ONE click.
  • Zotero - fabulously free way for students of life to collect, manage, and cite your research sources online.

Untested Firefox extensions for Developers

  • TinyUrl Creator - shrink long URLs into much shorter URLs that are easier to email or type in.
  • SeoQuake - obtain and investigate many important SEO parameters.
  • Firefox Accessibility - test web resources for accessibility features
  • ColorZilla - Advanced Eyedropper, ColorPicker, Page Zoomer and etc.
